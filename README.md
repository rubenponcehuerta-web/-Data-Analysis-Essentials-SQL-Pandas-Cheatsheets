# 📊 Data Analysis Essentials: Interactive Cheatsheets

Welcome to my Data Analysis repository! This project features a collection of high-performance, interactive cheatsheets designed to streamline the workflow of a **Junior-to-Mid Data Analyst**. 

Instead of static documentation, I developed these tools using **HTML/CSS/JS** to provide a fast, responsive, and "copy-paste ready" experience for the most critical concepts in SQL and Python.

---

## 📂 What's Inside?

### 1. 🐘 SQL: Junior to Mid-Level Mastery
Focuses on relational database logic and query optimization.
* **Core Syntax:** Deep dive into SELECT, WHERE, GROUP BY, and complex JOINS.
* **Advanced Analytics:** In-depth coverage of **Window Functions** (`RANK`, `ROW_NUMBER`, `LEAD/LAG`) and **CTEs** (`WITH` clauses) for modular and readable code.
* **Business Logic:** Practical examples of how to translate complex business questions into efficient SQL queries.

### 2. 🐼 Pandas: Data Manipulation & Cleaning
A comprehensive guide to the Python Data Science ecosystem, built for speed and clarity.
* **Data Wrangling:** Loading CSV/Excel files, handling missing values, and data type conversion.
* **Deep Analysis:** Grouping, pivoting, merging DataFrames, and advanced filtering using `.loc` and `.iloc`.
* **User Experience:** Custom-built "Dark Mode" UI with integrated code-highlighting to reduce eye strain during analysis sessions.

## 🛠️ Tech Stack & Key Features
* **HTML5 & CSS3:** Custom responsive grid layout with a modern developer aesthetic.
* **JavaScript:** Integrated **"Click-to-Copy"** functionality for all code snippets to boost productivity.
* **Documentation Standards:** Developed with a structured approach, mimicking professional software documentation and Agile workflows.

## 💡 Why I Built This
As I prepare for my first **Data Analyst** role, I realized that mastering tools like SQL and Pandas requires quick access to logic patterns. I built these guides to:
1. Synthesize complex technical documentation into actionable tools.
2. Demonstrate my ability to present technical data and concepts clearly to stakeholders.
3. Provide a practical bridge between theoretical learning and professional-grade execution.
