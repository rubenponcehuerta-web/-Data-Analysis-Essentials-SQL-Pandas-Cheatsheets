📊 Data Analysis Essentials: SQL & Pandas Cheatsheets
Welcome! This repository contains a collection of interactive guides (Cheatsheets) designed to streamline the daily workflow of a Data Analyst.

My goal in creating them was to synthesize key Junior to Mid-level concepts into visual, quick-reference tools.

🚀 What's included in these guides?
1. 🐘 SQL Junior to Mid-Level
A guide focused on relational database logic, covering everything from basics to advanced functions.

Key Concepts: Complex Joins, Aggregations (GROUP BY), and the use of Window Functions (RANK, ROW_NUMBER) and CTEs (WITH queries).

Extra: Optimization tips and business logic applications.

2. 🐼 Pandas for Data Science
Everything you need for data manipulation and cleaning with Python.

Content: Data loading, advanced filtering (loc/iloc), handling missing values, grouping, and merging DataFrames.

Visuals: Designed for quick "copy-paste" of code snippets.

🛠️ Technologies Used
HTML5 / CSS3: Responsive layout and "Dark Mode" design to reduce eye strain.

JavaScript: Interactive "click-to-copy" functions for efficient code retrieval.

💡 How to use them?
You can clone this repository and open the .html files in any browser, or view them directly via GitHub Pages.
